# 命名規則

以下のガイドラインに則った命名を行ってください。

## 基本ルール

一部の例外を除き、ディレクトリ名、ファイル名、および、HTML や CSS で使用する ID 名、クラス名の命名は以下を基本ルールとします。

- 英語を使用する
- 日本語のローマ字表記は原則禁止、やむを得ず使う場合はヘボン式を使用する
- 半角英数字と`-`（ハイフン）のみ使用する
- アルファベットからはじめ数字や`-`（ハイフン）からはじめない
- 末尾に `-`（ハイフン）で終わらない
- スペースは使用しない
- 連番数字は 2 桁以上のゼロ詰め表記を使用する

::: info 例外
以下は、上記のルールには該当しない例外です。

- 環境上必要な `.` からはじまる隠しファイルやディレクトリ （ `.htaccess` 、`.vscode` など ）
- デバイスピクセル比を識別する `@` を含んだ画像ファイル（`picture@2x.jpg` など）
- Sass などの `_` からはじまるパーシャルファイル （`_header.scss` など）
- CMS などにより動的に生成される数字のみのファイルやディレクトリ（ `123456` など ）

:::

## ディレクトリ名

内包するファイルのカテゴリを明確に表した英語を[ケバブケース](./glossary.html#ケバブケース)で命名してください。

## ファイル名

| ファイル                      | 命名規則                                           | 備考                                                                 |
| ----------------------------- | -------------------------------------------------- | -------------------------------------------------------------------- |
| HTML                          | [ケバブケース](./glossary.html#ケバブケース)   | ページのタイトルと同等の英語<br>（各ディレクトリの Top は `index` ） |
| CSS                           | [ケバブケース](./glossary.html#ケバブケース)   | コンポーネント名、モジュール名                                       |
| JavaScript （クラス定義以外） | [ケバブケース](./glossary.html#ケバブケース)   | モジュール名                                                         |
| JavaScript （クラス定義のみ） | [パスカルケース](./glossary.html#パスカルケース) | モジュール名                                                         |
| TypeScript （クラス定義以外） | [ケバブケース](./glossary.html#ケバブケース)   | モジュール名                                                         |
| TypeScript （クラス定義のみ   | [パスカルケース](./glossary.html#パスカルケース) | モジュール名                                                         |
| Astro （page）                | [ケバブケース](./glossary.html#ケバブケース)   | ページのタイトルと同等の英語<br>（各ディレクトリの Top は `index` ） |
| Astro （layouts、components） | [パスカルケース](./glossary.html#パスカルケース) | レイアウト名、コンポーネント名                                       |
| Vue                           | [パスカルケース](./glossary.html#パスカルケース) | コンポーネント名                                                     |
| メディア（画像、動画 など）   | [ケバブケース](./glossary.html#ケバブケース)   | コンポーネント名                                                            |


## コンポーネント名

コンポーネントの名前は、「種類」 + 「詳細」+ 「連番」 + 「状態」で構成し、利用する場所の命名規則（[ケバブケース](./glossary.html#ケバブケース)や、[パスカルケース](./glossary.html#パスカルケース) など）に合わせて連結します。

| 項目 | 説明                   | 備考                 |
| ---- | ---------------------- | -------------------- |
| 種類 | コンポーネントの種類名 | 必須                 |
| 詳細 | 利用用途や場所         | 必要に応じて         |
| 連番 | 2 – 3桁の ゼロ詰      | どうしても必要な場合 |
| 状態 | 状態や変化     | 特定の状態や変化を示す必要がある場合 |

### 種類

「種類」は必須で、コンポーネントの判断が出来る英単語を利用します。

他に同じ コンポーネントが存在しない（将来も含め）場合は、「種類」だけをコンポーネント名とします。

以下は主なコンポーネントの例です。






#### グルーピング

| コンポーネント     | 種類名      |
| :----------------- | :---------- |
| ページ             | `page`      |
| ヘッダー           | `header`    |
| フッター           | `footer`    |
| コンテナ | `container` |
| コンテンツグループ | `contents` |
| 章コンテンツ       | `section`   |
| 記事コンテンツ     | `article`   |
| 投稿コンテンツ     | `post`      |
| 汎用コンテンツ     | `content`   |

#### レイアウト

| コンポーネント | 種類名   |
| :------------- | :------- |
| グリッド       | `grid`   |
| スペーサー     | `spacer` |

#### パーツ

| コンポーネント     | 種類名    |
| :----------------- | :-------- |
| ナビゲーション     | `nav`     |
| メインビジュアル   | `hero`    |
| モーダルウインドウ | `modal`   |
| カルーセル | `carousel`   |
| カード             | `card`    |
| メディア           | `media`   |
| 検索               | `search`  |
| 地図               | `map`     |
| リスト               | `list`     |
| アイテム               | `item`     |
| 表              | `table`     |
| ロゴ               | `logo`    |
| タイトル           | `title`   |
| 見出し             | `heading` |
| 汎用テキスト       | `text`    |
| ボタン             | `button`  |

#### メディア

| コンポーネント     | 種類名    |
| :----------------- | :-------- |
| カバー画像         | `cover`   |
| サムネイル         | `thumb`   |
| バナー             | `banner`  |
| アイコン           | `icon`    |
| 背景               | `bg`      |
| 汎用画像           | `image`   |
| 汎用動画           | `video`   |
| 汎用音声           | `audio`   |

以下は CSS のクラス名に使用した例です。

```css
/* ブロック（種類） */
.header {
  display: block flex;
}

/* エレメント（種類__種類） */
.header__title {
  font-weight: 700;
}
```

### 詳細

「種類」だけではコンポーネントを識別できない場合は、「詳細」を連結します。

詳細は、利用用途や利用する場所など、シチュエーションが判断が出来る英単語を利用します。

以下は種類の例です。

| 利用用途や場所   | 種類名        |
| :--------------- | :------------ |
| 全体的           | `global`      |
| 局所的           | `local`       |
| 主要       | `main`     |
| サブ       | `sub`   |
| 根本             | `root`        |
| 枝               | `branch`      |
| パンくずナビ     | `breadcrumbs` |
| ページネーション | `pagination`  |

以下は CSS のクラス名に使用した例です。

```css
/* ブロック（種類+詳細） */
.nav-global {
  display: block flex;
}

/* エレメント（種類+詳細__種類+詳細） */
.nav-local__list-item {
  font-weight: 700;
}
```

### 連番

「連番」は基本的には使用しませんが、カルーセルなどでたくさんの画像ファイルなどで、「種類」+「詳細」だけではコンポーネントを識別できない場合は、「詳細」を連結します。

連番は、2 – 3桁のゼロ詰で表記します。


以下は 画像のファイル名に使用した例です。

```html
<!-- 種類+連番 -->
<img src="/assets/images/carousel-01.jpg" width="100" height="100" alt="..." />
<img src="/assets/images/carousel-02.jpg" width="100" height="100" alt="..." />
<img src="/assets/images/carousel-03.jpg" width="100" height="100" alt="..." />
```

以下は CSS のクラス名に使用した例です。

```css
/* エレメント（種類__種類+連番） */
.carousel__item-01 {
  margin-block-start: calc(24 * var(--torem));
}

/* エレメント（種類__種類+連番） */
.carousel__item-02 {
  margin-block-start: calc(32 * var(--torem));
}
```




### 状態

特定の状態の時のみの画像ファイルなどの場合には、最後に 「状態」を連結させます。

状態は、特定のUIの状態や、デバイスピクセル比、サイズなどのバリエーションを英単語で指定します。
なお、「詳細」や「連番」が必要な場合は、省略しても構いません。


状態は以下のようなものがあります。


#### UIの状態
| 状態   | 状態名        |
| :--------------- | :------------ |
| ホバー           | `is-hover`      |
| アクティブ           | `is-active`       |
| 表示       | `is-show`     |
| 非表示       | `is-hidden`   |

#### デバイスピクセル比
| デバイスピクセル比  | 状態名        |
| :--------------- | :------------ |
| 2           | `@2x`      |
| 3          | `@3x`       |

#### サイズ
| サイズ  | 状態名        |
| :--------------- | :------------ |
| 小          | `small`      |
| 中           | `medium`       |
| 大       | `large`     |



```html
<!-- 種類+連番+状態 -->
<img src="/assets/images/carousel-01.jpg" src="/assets/images/carousel-01@2x.jpg" width="100" height="100" alt="..." />
<img src="/assets/images/carousel-02.jpg" src="/assets/images/carousel-02@2x.jpg" width="100" height="100" alt="..." />
<img src="/assets/images/carousel-03.jpg" src="/assets/images/carousel-03@2x.jpg" width="100" height="100" alt="..." />
```
